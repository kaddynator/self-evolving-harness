# Gemini Integration Skill

## Purpose
Integrate Gemini models and managed agents.

## When to use
Use when adding Gemini calls for organization synthesis, prompt generation, mutation proposals, or agent execution.

## Instructions
1. Keep Gemini behind an adapter interface.
2. Support mock mode for reliable demos.
3. Use Gemini for:
   - compiling organizations
   - generating prompts
   - proposing mutations
   - summarizing lessons
   - executing agent roles
4. Never make demo success depend entirely on live model randomness.
5. Cache useful outputs where possible.
