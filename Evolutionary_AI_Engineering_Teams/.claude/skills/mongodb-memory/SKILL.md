# MongoDB Memory Skill

## Purpose
Implement or modify persistent memory.

## When to use
Use when working on MongoDB schemas, run storage, lessons, or organization history.

## Instructions
1. Store every task, organization, run, evaluation, mutation, and lesson.
2. Preserve parent-child relationships between organization versions.
3. Make it easy to query:
   - best organization for a task
   - all mutations
   - run metrics over time
   - learned lessons
4. Keep documents simple for hackathon demo.

## Suggested collections
- tasks
- organizations
- runs
- evaluations
- mutations
- lessons
