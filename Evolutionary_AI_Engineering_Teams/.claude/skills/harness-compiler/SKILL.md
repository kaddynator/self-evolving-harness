# Team / Harness Compiler Skill

## Purpose
Compile a task into a complete Organization Harness IR.

## Instructions
1. Generate not just agents, but the full harness:
   - agents
   - prompts
   - tools
   - communication graph
   - runtime policies
   - failure recovery
   - evaluation metrics
2. Use prior MongoDB lessons when available.
3. Output valid Organization Harness IR.
4. Keep the first version simple and executable.

## MVP
Start from templates, then allow mutation and later Gemini synthesis.
