Evolutionary AI Engineering Teams

Core Architecture

Task Specification ↓ Team Compiler - Creates agents - Assigns prompts -
Assigns tools - Defines communication graph ↓ Runtime - Executes agents
inside isolated sandboxes ↓ Evaluator - Scores task success - Measures
tool calls - Measures runtime - Measures code quality ↓ Evolution
Engine - Adds agents - Removes agents - Updates prompts - Changes tool
permissions - Changes execution order ↓ Memory (MongoDB) - Stores
organization versions - Stores evaluation scores - Stores lessons
learned - Stores successful organization patterns

------------------------------------------------------------------------

Recommended Demo Task

Implement a small software engineering feature in a deterministic
repository.

Example: - Build a rate limiter for an API endpoint. - Existing tests
must continue passing. - New functionality must satisfy acceptance
criteria.
